# wales_ish_r5r_runner
This repository produces travel time matrices for journeys between 
[LSOA Trip-points](https://github.com/stupidpupil/wales_lsoa_trip_points) 
using [this set of public transport and street map data](https://stupidpupil.github.io/wales_ish_otp_graph/).
